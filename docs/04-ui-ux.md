# 04 - UI/UX y diseño visual

## Dirección visual
Profesional, elegante, sobria y moderna.

## Tema oscuro
- Fondo principal: azul/negro profundo.
- Superficies: gris azulado oscuro.
- Texto principal: blanco suave.
- Texto secundario: gris claro.
- Acentos: azul, cian o violeta sobrio.
- Estados: verde, amarillo, rojo, gris.

## Tema claro
- Fondo principal: gris muy claro.
- Superficies: blanco.
- Texto principal: gris/azul oscuro.
- Mantener los mismos acentos y estados.

## Layout
- Sidebar contraíble.
- Topbar con búsqueda, notificaciones, theme toggle y usuario.
- Contenido con filtros arriba, KPIs, gráficos y tabla.

## Ver detalle
Usar drawer lateral o modal grande. No perder el contexto de la vista actual.

## Componentes
- KPICard
- ChartCard
- DataTable
- FiltersBar
- StatusBadge
- ExportButton
- DrawerDetail
- ConfirmDialog
- EmptyState
- LoadingSkeleton

## Accesibilidad
- Contraste suficiente en ambos temas.
- Estados no deben depender solo del color; usar texto e iconos.
- Tablas legibles.
- Botones con labels claros.
